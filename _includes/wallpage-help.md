# 安卓與iPhone動態桌紙設定速成指南

## 下載準備

在本網站選擇喜愛的動態桌紙後：

- **安卓用戶**：下載MP4格式影片檔

- **iPhone用戶**：下載Live Photo實況格式（含.mov與.jpg檔案）

## 安卓設定步驟（影片檔）

1. **下載完成後**，進入手機「檔案管理」
2. **找到下載的影片檔**，長按選擇「設為桌布」
3. **或使用第三方應用**：安裝「Video Wallpaper」應用程式，從應用內選擇本網站下載的影片檔
4. **調整設定**：建議開啟循環播放，可選擇靜音節省耗電
5. **套用完成**：選擇設為主畫面或鎖定畫面

## iPhone設定步驟（實況格式）

1. **匯入檔案**：將下載的.mov和.jpg檔案透過iCloud或AirDrop傳到iPhone
2. **開啟「照片」應用**：找到匯入的實況照片
3. **點擊分享圖示** → 選擇「用作背景圖片」
4. **調整位置**後點擊「設定」
5. **重要**：進入「設定」>「背景圖片」，確認「原況照片」選項已開啟
6. **鎖定畫面下重壓螢幕**即可看到動態效果

## 效能提示

- 旗艦機型可選4K畫質，舊機型建議使用1080p
- 低電量時建議暫停使用動態桌紙
- 深色系桌紙較節省電力（特別適合OLED螢幕）
